增长运营 — 平台新老用户
负责人：Naval + 增长团队 · 覆盖站内所有注册用户
方案 01
充值阶梯奖励
拉升首月平台资金存量，激励用户追加充值探索预测市场
  • 充值 $100 赠 $10 USDT 体验金
  • 充值 $500 赠 $30 USDT 体验金
  • 充值 $1,000+ 赠 $80 USDT 体验金
  • 体验金 7 天有效，仅限预测市场使用
  • 站内 Push + 邮件 + Banner 全渠道触达
KPI：活动期新增充值用户 ≥500，人均充值 ≥$300
方案 02
新用户双礼入场
降低新用户首次尝试门槛，提高 D-0 注册转化率
  • 首次注册赠 $10 预测市场体验金
  • 完成 KYC 后额外解锁 3 笔免费预测额度（max $10/笔）
  • 邀请码专属通道，邀请人同步获返佣
  • 新人 7 日成长任务（首次预测 → 首次出金）
  • 完成全任务额外奖励 $5
KPI：上线首周新注册 ≥1,000，7 日留存率 ≥40%
方案 03
0% Fee Launch + 交易量排行榜
以永久 0% 手续费席位为核心钩子，驱动重度用户高频交易
  • D-0 宣布预测市场 0% 手续费（竞品 1–10%）
  • 交易量 $500 返 $20，$1,000 返 $50，$3,000 返 $150
  • 月度交易量 Top 10 永久锁定 0% 席位
  • 实时排行榜每 4 小时刷新，公开展示
  • 数据出口阈值：DAU ≥500 且交易量 ≥$1M
KPI：月活跃交易用户 ≥500，月交易量 ≥$1M
苍苍组 — 欧美 KOL 代理
负责人：苍苍 · 覆盖英文市场 KOL 生态
方案 01
上线日 KOL 矩阵内容
D-0 形成英文市场声量爆发，多账号同步触发算法推流
  • D-14 发送 KOL Briefing 包（内容方向 + 素材 + 话术）
  • 确认 10+ KOL 参与，分层设定发布时间窗口
  • D-0 10:00 UTC 起统一推文，话题标签护航
  • Tier 1 视频深度介绍，Tier 2/3 短文推广
  • 引导评论区互动，提升算法权重
KPI：上线日英文推文覆盖 ≥500K 展示量
方案 02
KOL 带头预测挑战
通过 KOL 公开下注引发跟风效应，展示预测市场真实玩法
  • 邀请 3–5 名头部 KOL 公开下注热门预测题
  • KOL 全程记录（买入 → 等待 → 结算）
  • 结果揭晓后晒单，带 $MOONX 话题二次传播
  • 胜出 KOL 获专属徽章 + 平台首页展示位
KPI：挑战帖互动 ≥50K，带动新增用户 ≥200
方案 03
KOL 专属邀请返佣
将 KOL 影响力转化为持续性注册来源，建立长期代理关系
  • 每位 KOL 分配专属邀请链接 + 落地页
  • 邀请注册用户完成首充即触发返佣
  • 阶梯返佣：10 人→$50，50 人→$300，100 人→$800
  • 月度 Top 3 代理额外奖励 + 平台认证标识
KPI：欧美区邀请注册 ≥500，首充转化率 ≥30%
Sally 组 — 韩国 KOL 代理
负责人：Sally · 覆盖韩语市场 KOL 生态
方案 01
韩文上线日内容矩阵
在韩国加密社区形成集中声量，建立本地化认知
  • D-14 发送韩文 KOL Briefing（本地化内容方向）
  • 提供韩语话术模板、平台截图素材
  • D-0 配合欧美 KOL 同步，按韩国时区优化排期
  • Kakao / Twitter / YouTube Shorts 全渠道覆盖
KPI：上线日韩语内容覆盖 ≥200K 展示量
方案 02
韩国预测挑战 + 排行榜
激活韩国用户竞技属性，以独立排行榜形成本地竞争氛围
  • 设立韩区专属预测挑战题（加密热点话题）
  • 独立韩语排行榜，Top 10 韩区用户赢专属奖励
  • KOL 担任"挑战发起人"，公开参与引流
  • 结果揭晓后 KOL 直播开奖，带动二次传播
KPI：韩区参与用户 ≥300，韩语帖子互动 ≥20K
方案 03
裂变邀请（韩语版）
通过韩国用户社交网络裂变，低成本获取本地精准用户
  • 韩语版邀请落地页，本地化 UI 与文案
  • 邀请阶梯奖励同欧美版，韩语通知设计
  • Kakao Talk 群组专属邀请码，方便社群传播
  • 韩区月度 Top 代理额外奖励 + KOL 认证
KPI：韩区邀请注册 ≥300，裂变系数 ≥1.5
Michael 组 — 媒体代理
负责人：Michael · 覆盖 T1/T2 加密媒体渠道
方案 01
T1/T2 媒体稿件 + PR Wire
上线日在主流加密媒体形成报道覆盖，建立机构公信力
  • D-14 锁定媒体名单（CoinDesk / Decrypt / The Block 等）
  • D-7 提交稿件初稿，配合编辑审核节奏
  • D-0 PR Wire 全球同步发布
  • 核心卖点：0% 手续费 + 预测市场数据可信度
KPI：上线日 T1/T2 报道 ≥5 篇，总覆盖读者 ≥1M
方案 02
媒体资源包 + Pitch
为媒体代理提供标准化素材包，降低二次传播门槛
  • 制作资源包：Logo / 截图 / 数据图表 / 角色海报
  • 准备 3 版 Pitch（长文 / 短讯 / 社媒摘要）
  • 提供平台独家数据 Embargo 权限（D-1 提前解锁）
  • 专属媒体联系人直线，响应时间 ≤4h
KPI：资源包下载 ≥20 家，独家报道 ≥2 篇
方案 03
代理冲榜月
激励媒体代理持续产出报道，形成长尾媒体覆盖效应
  • 上线后 30 天设立媒体代理排行榜
  • 按报道数量 + 阅读量加权计分
  • Top 3：$1,000 / $600 / $300 现金奖励
  • 月底公示排名，优质稿件平台官方转载
KPI：30 天内累计报道 ≥30 篇，总阅读量 ≥5M
Chloe — 外部大媒体宣发
内部员工 · 负责与外部大媒体直接合作，覆盖高权重渠道
方案 01
上线日大媒体联合报道
与 1–2 家头部媒体达成独家/联合报道，形成权威背书
  • D-21 接触目标大媒体（CoinTelegraph / Forbes Crypto 等）
  • 提供 CEO 独家专访作为合作筹码
  • D-0 同步发布，标注独家首发权益
  • 合作媒体在其 Twitter / Newsletter 同步推送
KPI：大媒体独家报道 ≥1 篇，单篇阅读量 ≥100K
方案 02
活动期媒体跟进曝光
活动期间持续提供数据节点，保持媒体持续关注度
  • D+7 提供充值活动首周数据给已合作媒体
  • D+14 提供交易量排行榜高亮数据
  • 每次数据节点撰写 200 字简讯推送媒体
  • 出现黑马用户/话题，快速协调媒体跟进
KPI：活动期追加报道 ≥3 篇，媒体热度不断档
方案 03
行业媒体月度成绩单
D+30 以数据化成绩单形式发布上线总结，巩固行业地位
  • D+28 向 Nate 获取完整月度数据
  • 撰写图文版成绩单（注册量 / 交易量 / 媒体覆盖等）
  • 联系 3+ 家行业媒体同步发布
  • 成绩单同步在平台官方社媒发布
KPI：覆盖媒体 ≥3 家，行业曝光 ≥500K
SEO 组 — Vitalik
独立团队 · 覆盖搜索自然流量 + 注册转化漏斗
方案 01
0 费率 SEO 文章 + 落地页
捕获"预测市场 0 手续费"搜索意图，转化高质量直客
  • 核心词：prediction market 0 fee / best prediction market
  • 竞品对比文章（PredictIt 10% vs MoonX 0%）
  • 专属落地页：0% Fee 费率计算器 + 注册 CTA
  • D-7 文章上线，D-0 配合声量推送
KPI：30 天自然流量 ≥5K UV，落地页注册转化率 ≥8%
方案 02
注册转化漏斗 5 步
系统化消除注册到首次预测的摩擦点，提升全链路转化
  • 步骤 1：SEO 文章 → 点击 CTA → 落地页
  • 步骤 2：落地页 → 注册（邮件 + 密码，简化流程）
  • 步骤 3：注册完成 → 新手引导（体验金激活）
  • 步骤 4：首次预测 → 结果通知（Push / Email）
  • 步骤 5：结算 → 出金引导 → 二次预测钩子
KPI：全漏斗转化率 ≥5%（落地页访问 → 首次预测）
方案 03
7 封激活邮件序列
对注册未充值用户进行 7 天自动化激活，提升首充转化
  • D+0（注册当天）：欢迎 + 体验金领取引导
  • D+1：0% 费率对比图 + 首个热门预测题推荐
  • D+3：充值阶梯奖励提醒（$100 赠 $10）
  • D+5：KOL 挑战晒单 + 真实用户故事
  • D+7：最终召回（体验金即将到期警示）
KPI：7 日首充转化率 ≥15%，邮件打开率 ≥35%
